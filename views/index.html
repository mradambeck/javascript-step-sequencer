[ bleep bloop ]

what's this?

Bleep/Bloop is a step sequencer programmed in Javascript.

Using the Web Audio Api, this is a synthesizer programmed to go through a sequence of notes, allowing you to create repeating melodies and patterns.

As a musician, when I first heard that Javascript natively included an audio feature capable of creating your own Oscillators, I knew I had to get my hands on it. It's been a fun project, as it presented a number of problems you wouldn't usually come across in web development, which are specific to creating an instrument.
